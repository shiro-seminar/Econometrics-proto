# 金融政策アナウンスメントと学費値上げの市場・受験行動への影響分析

## 概要

本リポジトリは2つの計量経済学的分析を含みます。

1. **イベントスタディ**：日銀・FOMCの政策変更アナウンスメントが日経平均・ドル円に与える影響
2. **差分の差分法（DiD）**：国立・私立大学の学費値上げが志願者数に与える影響

---

## 分析1：イベントスタディ（`event_study.ipynb`）

### データ
- 日経平均（`^N225`）・ドル円（`JPY=X`）：yfinance経由で取得
- 日銀政策変更イベント4件（2024〜2025年）
- FOMCイベント3件（2024年）

### 手法
- 正常リターンの推定：イベント前[-120, -10]営業日の平均リターン
- 異常リターン（AR）= 実際のリターン - 正常リターン
- 累積異常リターン（CAR）の計算
- t検定によるCARの統計的有意性の検証

### 主な結果
- 2024年7月31日の日銀利上げのみ統計的に有意（CAR=-11.8%, p=0.001）
- 他のイベントは事前に市場に織り込まれており有意な影響なし
- 緩和イベント全般では平均CARが正で有意（p=0.036）

## 分析2：DiD分析（`did_tuition.ipynb`）

### データ
- 国立・私立大学8校の志願者数（代々木ゼミナール・各大学HP等）
- 期間：2015〜2025年
- 処置群：東大・東工大・一橋・東京藝大・早稲田・慶應（学費値上げ実施）
- 対照群：京大・名大・九大（標準額維持）

### 手法
- 大学固定効果・年度固定効果付きDiD（HC1ロバスト標準誤差）
- イベントスタディ型DiDによる年次効果の推定

### 主な結果
- 学費値上げ後1〜2年間は志願者数が約5〜7%減少する傾向
- ただし統計的有意水準には達しない（サンプルサイズの制約）
- 平行トレンド仮定は事前期間で確認済み
- 3〜4年後には効果が消滅し回復傾向

### 分析の限界
- 値上げ事例が限られるため統計的検出力が低い
- 対照群が3大学と少ない
- 今後値上げ大学が増えれば再推定可能な設計
